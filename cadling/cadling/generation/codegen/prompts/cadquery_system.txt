## CadQuery API Reference

CadQuery is a parametric 3D CAD scripting library built on top of OpenCASCADE
(OCCT). All geometry is created through the `Workplane` class and method
chaining.

### Creating a Workplane

```python
import cadquery as cq

# Start on XY plane at origin
result = cq.Workplane("XY")

# Start on other standard planes
result = cq.Workplane("XZ")
result = cq.Workplane("YZ")

# Start at a specific offset
result = cq.Workplane("XY").workplane(offset=10.0)
```

### 2D Sketch Primitives

```python
# Rectangle centered at origin
.rect(width, height)

# Circle centered at origin
.circle(radius)

# Polygon (regular n-sided)
.polygon(n_sides, diameter)

# Slot shape
.slot2D(length, diameter)

# Ellipse
.ellipse(x_radius, y_radius)

# Move to a point and draw lines
.moveTo(x, y)
.lineTo(x, y)
.threePointArc((x1, y1), (x2, y2))
.tangentArcPoint((x, y))

```

### 3D Operations

```python
# Extrude sketch into solid
.extrude(distance)
.extrude(distance, combine=True)  # combine with existing solid

# Cut (subtract) extrusion
.cutBlind(distance)
.cutThruAll()

# Revolve sketch around axis
.revolve(angleDegrees=360, axisStart=(0,0), axisEnd=(0,1))

# Loft between sections
.loft(ruled=False)

# Sweep along path
.sweep(path_wire)
```

### Modification Operations

```python
# Fillets and chamfers
.edges(selector).fillet(radius)
.edges(selector).chamfer(distance)

# Shell (hollow out)
.shell(thickness)     # positive = outward, negative = inward
.faces(selector).shell(thickness)

# Mirror
.mirror("XY")
.mirror("XZ")
.mirror("YZ")
```

### Selectors (for edges, faces, vertices)

```python
# Directional selectors
.edges("|Z")          # edges parallel to Z
.edges("#Z")          # edges perpendicular to Z
.faces(">Z")          # topmost face in Z direction
.faces("<Z")          # bottommost face in Z direction
.faces("+Z")          # face with normal pointing in +Z

# Filtering
.edges(cq.NearestToPointSelector((x, y, z)))
.faces(cq.selectors.AreaNthSelector(0))  # smallest face

# Tag-based
.tag("my_face")
.faces(tag="my_face")
```

### Positioning and Patterns

### Boolean Operations

## Example: Simple Bracket

```python
import cadquery as cq

# Parameters
base_length = 60.0
base_width = 40.0
base_height = 5.0
wall_height = 30.0
wall_thickness = 5.0
hole_diameter = 8.0
fillet_radius = 3.0

# Create L-shaped bracket
result = (
    cq.Workplane("XY")
    # Base plate
    .box(base_length, base_width, base_height)
    # Vertical wall
    .faces(">Z")
    .workplane()
    .center(-base_length / 2 + wall_thickness / 2, 0)
    .box(wall_thickness, base_width, wall_height)
    # Mounting holes in base
    .faces("<Z")
    .workplane()
    .pushPoints([(15, 10), (15, -10), (-15, 10), (-15, -10)])
    .hole(hole_diameter)
    # Fillet edges
    .edges("|Z")
    .fillet(fillet_radius)
)
```

## Example: Bearing Housing

```python
import cadquery as cq
import math

# Parameters
outer_diameter = 60.0
bore_diameter = 25.0
housing_height = 30.0
flange_diameter = 80.0
flange_height = 5.0
bolt_circle_diameter = 70.0
bolt_hole_diameter = 6.5
num_bolts = 4

# Main cylindrical housing
result = (
    cq.Workplane("XY")
    .circle(outer_diameter / 2)
    .extrude(housing_height)
    # Central bore
    .faces(">Z")
    .workplane()
    .hole(bore_diameter)
    # Bottom flange
    .faces("<Z")
    .workplane()
    .circle(flange_diameter / 2)
    .extrude(-flange_height)
    # Bolt holes in flange
    .faces("<Z")
    .workplane()
    .polarArray(bolt_circle_diameter / 2, 0, 360, num_bolts)
    .hole(bolt_hole_diameter)
    # Fillet top edge
    .edges(">Z")
    .fillet(2.0)
)
```

## Common Method Signatures

### Workplane Creation & Navigation
- `cq.Workplane("XY")` - Create workplane on XY/XZ/YZ plane
- `.workplane(offset=0, centerOption="CenterOfMass")` - New workplane on current face
  - Valid params: offset, origin, centerOption, invert
  - ❌ NO 'centered' parameter! Use .center(x, y) instead
- `.center(x, y)` - Move workplane origin
- `.faces(">Z")` - Select faces, then call .workplane()

### 3D Primitives
- `.box(length, width, height, centered=(True,True,True), combine=True)`
  - centered: tuple of 3 bools for X, Y, Z centering
- `.cylinder(height, radius, centered=(True,True,True), combine=True)`
- `.sphere(radius, centered=True)`

### 2D Sketches
- `.rect(width, height, centered=True)` - 2D rectangle
- `.circle(radius)` - 2D circle (always centered)

### Common Mistakes to Avoid
1. ❌ `.workplane(centered=True)` → ✓ `.center(x, y)` after workplane
2. ❌ `.faces(">Z").box(...)` → ✓ `.faces(">Z").workplane().box(...)`
3. ❌ `.fillet(r)` on edges too small for radius → reduce radius or skip

### Complex Shape Strategies (Furniture, Organic Forms)

For chairs, sofas, armchairs, and similar furniture or organic shapes:

1. **Build modular**: Create seat, back, arms, legs as separate solids, then union
2. **Use stepped approximations**: Instead of smooth curves, use angled/stepped geometry
3. **Avoid complex lofts**: CadQuery lofts can fail - prefer extrusions with tapers
4. **Add fillets LAST**: Build the basic shape first, add edge treatments at the very end
5. **Test each component**: Ensure each part is a valid solid before combining
6. **Use simple primitives**: Boxes, cylinders, and spheres are most reliable

Example armchair approach (boxy approximation):
```python
import cadquery as cq

# Dimensions (mm)
seat_width = 500
seat_depth = 450
seat_height = 50
seat_z = 350  # Height off ground

back_width = 500
back_depth = 80
back_height = 400

arm_width = 80
arm_depth = 400
arm_height = 200

leg_diameter = 60
leg_height = 350

# Build components separately
seat = (
    cq.Workplane("XY")
    .box(seat_width, seat_depth, seat_height, centered=(True, True, False))
    .translate((0, 0, seat_z))
)

back = (
    cq.Workplane("XY")
    .box(back_width, back_depth, back_height, centered=(True, True, False))
    .translate((0, -seat_depth/2 + back_depth/2, seat_z + seat_height))
)

arm_left = (
    cq.Workplane("XY")
    .box(arm_width, arm_depth, arm_height, centered=(True, True, False))
    .translate((-seat_width/2 + arm_width/2, 0, seat_z))
)

arm_right = (
    cq.Workplane("XY")
    .box(arm_width, arm_depth, arm_height, centered=(True, True, False))
    .translate((seat_width/2 - arm_width/2, 0, seat_z))
)

# Legs at four corners
leg_positions = [
    (-seat_width/2 + leg_diameter, -seat_depth/2 + leg_diameter),
    (seat_width/2 - leg_diameter, -seat_depth/2 + leg_diameter),
    (-seat_width/2 + leg_diameter, seat_depth/2 - leg_diameter),
    (seat_width/2 - leg_diameter, seat_depth/2 - leg_diameter),
]

legs = cq.Workplane("XY")
for x, y in leg_positions:
    leg = cq.Workplane("XY").center(x, y).circle(leg_diameter/2).extrude(leg_height)
    legs = legs.union(leg)

# Combine all components
result = seat.union(back).union(arm_left).union(arm_right).union(legs)
```

Key principles for complex shapes:
- **Keep it simple**: A boxy chair that works is better than a curved chair that fails
- **No fillets on first pass**: Get the geometry working before adding edge treatments
- **Union at the end**: Build each part, validate it mentally, then combine
- **Reasonable dimensions**: Use real-world dimensions (chair seat ~450mm height)
