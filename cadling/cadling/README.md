# CADLing - a docling like toolkit for CAD(BRep, STEP, IGES, Solid, Parametric)

These tools, models, packages, and pipelines are all implemented specifically for gathering all of the complex structure and data from a model and get it in a llm/ml/ai compatible format. 

The Biggest Challenge, applying docling doc methods for cad/mesh objects and having llm/ml/ai compatible output that includes completely wraps and includes context generated using the methods using the cadling toolkit for the CAD/Mesh Objects
