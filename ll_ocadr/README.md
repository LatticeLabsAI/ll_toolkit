# Lattice Labs Optical CAD Recognition 

A DeepSeek-OCR inspired and recreated for CAD/Mesh using the same tiled chunks hi resolution and Global Full Resolution Object that the vision transformer applies context to and embeds both 

A DEEPSEEK-OCR inspired, and expanded and completely new usecase for their implementation except it doesnt process documents and images but instead processes CAD(STEP) Mesh(STL/OBJ) files to apply context on a token level using the Tiled Chunks(chunked sections of a cad/mesh object) and the Global Full Resolution CAD/Mesh Object and combines them and embeds the tokens ect.
